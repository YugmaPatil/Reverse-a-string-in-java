# Reverse-a-string-in-java
-Original String 
   "Hello"

-Reversed String 
    "olleH"

  # By using a for loop
  - str.length() - 1  --> Starts at index 4
  - The loop continues while i>=0, decrementing i each time
  - str.charAt(i) extracts one character at a time
  - reverse += ..... appends each character to build the reversed string
